This was a group project completed in ITCS 3162: Data Mining at UNC Charlotte. The overall goal of the project was to figure out what caused high housing prices in
California neighborhoods in 1990 and figure out how to predict housing prices. To conduct our data analysis, we used a software known as KNIME. Through this, instead of having to code everything out in Python and
in R, we were able to simplify our methods to obtain quick and easy statistics.

Overall, this project taught me a lot about using large datasets. We had many challenges like dealing with missing data in some datapoints, 
figuring out what type of method we wanted to use to analyze the data, and we had to figure out the data's limitations. A few examples of this 
are employment opportunties per region, inflation rate, interest rates, and crime. In the end, we decided that a linear regression was the best way 
to predict the housing and identify what made housing higher cost. Rooms per home and location tended to give the best predictor of housing prices.
